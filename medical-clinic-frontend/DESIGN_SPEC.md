# Medical Clinic Management System - Design Specification

## Design Philosophy
**Modern, Clean, and Intuitive** - A professional healthcare interface that prioritizes clarity, accessibility, and user confidence.

## Color Palette
- **Primary Blue**: `#0066CC` (Medical Trust & Professionalism)
- **Light Blue**: `#E8F4FD` (Calming, Approachable)
- **White**: `#FFFFFF` (Clean, Spacious)
- **Dark Gray**: `#1F2937` (Text, Hierarchy)
- **Light Gray**: `#F3F4F6` (Backgrounds, Dividers)
- **Success Green**: `#10B981` (Positive Actions)
- **Warning Orange**: `#F59E0B` (Alerts)
- **Error Red**: `#EF4444` (Destructive Actions)

## Typography
- **Display Font**: System font stack (clean, professional)
- **Body Font**: System font stack (readable, accessible)
- **Font Weights**: 400 (regular), 500 (medium), 600 (semibold), 700 (bold)

## Layout Principles
- **Sidebar Navigation**: For authenticated users (admin, doctor, patient dashboards)
- **Top Navigation**: For public pages (login, registration)
- **Responsive Grid**: Mobile-first approach with breakpoints at 640px, 1024px, 1280px
- **Spacing System**: 4px base unit (4, 8, 12, 16, 24, 32, 48, 64px)

## Component Style
- **Cards**: White background, subtle shadow, rounded corners (8px)
- **Buttons**: Primary (Blue), Secondary (Light Gray), Outlined
- **Forms**: Clean input fields with light borders, focus states with blue outline
- **Icons**: Lucide React (consistent, minimal)
- **Modals**: Centered, semi-transparent backdrop, smooth animations

## Accessibility
- WCAG 2.1 AA compliance
- Keyboard navigation support
- High contrast text (4.5:1 minimum)
- Clear focus indicators
- Semantic HTML structure

## Animation & Micro-interactions
- Smooth transitions (200-300ms)
- Hover effects on interactive elements
- Loading states with spinners
- Toast notifications for feedback
- Page transitions with fade effects
